# Export Files

---

## Export Teq4 and Teq4z to Excel

1. Click the Save icon.
   <Save className="w-10 h-10" />
2. Select the teq4 or teq4z tab.
   <div className="grid lg:grid-cols-2">
     <img src={ExportTeq4} alt="teq4" />
     <img src={ExportTeq4z} alt="teq4z" />
   </div>
3. Choose the files you want to export by selecting them individually, the name, the columns and isSameSheet option.
4. Click Export and select the path to save the file.
