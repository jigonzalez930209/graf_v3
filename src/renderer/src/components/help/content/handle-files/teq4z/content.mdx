# Teq4z

---

Teq4z files are natively supported by the Graf application

## How can handle Teq4Z files

You can select the file teq4z in the lateral menu and the file will be loaded in the main side.

Using the settings button, you can customize the color, width, and impedance type.

## How to export Teq4 files

- Click the Save icon.
  {' '}

  <Save className="w-5 h-5" />

- Select the TEQ4 tab.
  {' '}

  <img src={ExportTeq4z} alt="Teq4 options" className="col-span-2" />{' '}

- Choose the teq4z files you want to export by selecting them individually, the columns, the name and isSameSheet option.

- Click Export to select the path to export all selected files.
